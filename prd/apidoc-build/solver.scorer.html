Scorer module documentation¶

This module provides a Scorer class that will enable one to get the score of a configuration on a Geography.

Computing a score is the first necessary step to find the configuration that gives the lowest score (i.e. cost) for a given Geography.

Class defining a scorer (utility to compute score)

This class implements a scorer utility that will compute the score of a configuration for a geography.

A scorer must have a scoring function defined as an attribute, as well as a Geography.
